##  Blinding attack

Этот тип атаки позволяет получить действительную подпись на любом сообщении, попросив систему подписать "случайное" сообщение. При этом у системы нет информации о том, какое сообщение она фактически подписывает. Атака работает в схемах, в которых подписывающая сторона подписывает непосредственно само сообщение.
